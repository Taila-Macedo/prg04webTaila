
# 🧭 Horizonte Meu — Seu Atlas de Sonhos Interativo

Plataforma interativa para o mapeamento de metas e sonhos de viagem. Explore destinos globais através de um mapa dinâmico, crie roteiros turísticos reais e monte uma lista de desejos completamente personalizada.


---

## Sobre o Projeto

O **Horizonte Meu** nasceu da ideia de transformar sonhos de viagem em planos reais. A plataforma conecta o usuário a destinos do mundo inteiro, permitindo que ele explore, planeje e organize suas próximas aventuras de forma simples e visual.

---

## Estrutura

```
horizonte-meu/
│
├── infrastructure/
│   └── assets/
│       ├── css/
│       ├── fonts/
│       ├── images/
│       └── js/
│
├── pages/
│   ├── mapa/
│   ├── destinos/
│   ├── wishlist/
│   └── sobre/
│
├── index.html
└── README.md
```

### infrastructure/assets

| Diretório | Responsabilidade |
|-----------|-----------------|
| `css/` | Estilização e identidade visual da aplicação |
| `fonts/` | Tipografias utilizadas no projeto |
| `images/` | Imagens, ícones e favicon |
| `js/` | Lógica de interação e comportamento das páginas |

### pages

| Diretório | Responsabilidade |
|-----------|-----------------|
| `mapa/` | Mapa dinâmico para exploração de destinos globais |
| `destinos/` | Detalhes e roteiros de cada destino disponível |
| `wishlist/` | Lista de sonhos personalizada por usuário |
| `sobre/` | Apresentação e contexto do projeto |

> `index.html` — ponto de entrada da aplicação.  
> `README.md` — documentação do projeto.
