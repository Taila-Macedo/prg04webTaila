# prg04webTaila
🧭 Horizonte Meu - Seu Atlas de Sonhos Interativo

O Horizonte Meu é um projeto de plataforma web que transforma o planejamento de viagens e metas pessoais em uma experiência visual e interativa. Através de um mapa-múndi explorável, o usuário pode descobrir destinos, conhecer atrações turísticas icônicas e organizar sua própria lista de desejos para o futuro.

🚀 Funcionalidades Principais

Exploração por Mapa: Navegação intuitiva através de pins clicáveis em locais estratégicos ao redor do globo.

Cards de Destino: Informações detalhadas sobre a "vibe" de cada país, curiosidades culturais e principais pontos turísticos.

Integração com Google Maps: Conexão direta com a localização real das atrações para facilitar o planejamento.

Wishlist Dinâmica: Sistema de "Lista de Desejos" com feedback visual (notificações) ao adicionar novos sonhos.
